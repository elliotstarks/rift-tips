## Development

[We use Bridgetown](https://www.bridgetownrb.com/docs/)

```sh
git clone https://github.com/unRARed/rift-tips
cd rift-tips
bundle install && yarn install
```

Run `bin/bridgetown start` and navigate to [localhost:3000](https://localhost:3000/)!
